# Quack-Session  
BCS 2010 Reference

## What this is
A lightweight study workflow for working with:
- Section-based quiz JSON files  
- Referenced images/figures  
- Focused study sessions (report or practice)

## How to use (quick)
1. **Upload the instructions JSON**  
   Sets the rules for the session.

2. **Upload images (if any)**  
   Figures used by questions (e.g., `group1.jpg`, `isomers3.jpg`).

3. **Upload section JSON(s)**  
   The quiz data for the section(s) you want to study.

4. **Pick your focus areas**  
   Examples: functional groups, isomers, acids/bases, image questions.

5. **Start studying (default = report)**  
   Youâ€™ll get a short study report.  
   You can switch to:
   - one question at a time  
   - quick review  
   - flashcards  
   - exam-style practice

## What you get
- Clean, section-based study sessions  
- Image-aware questions  
- Optional progress summaries  
- Flexible study modes

## Tips
- Upload images before quiz data so figures can be referenced correctly.  
- Study one section at a time for best focus.  
- Ask for a progress summary anytime.
